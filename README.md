# 🚀 RustCoin - ارز دیجیتال ایرانی

یک پیاده‌سازی کامل از بلاک چین و ارز دیجیتال با استفاده از زبان برنامه‌نویسی Rust.

## ✨ ویژگی‌ها

- 🔗 **بلاک چین کامل**: پیاده‌سازی زنجیره بلاک با Proof of Work
- 💰 **سیستم تراکنش**: امضای دیجیتال با الگوریتم ECDSA
- ⛏️ **استخراج**: سیستم استخراج با قابلیت تنظیم سختی
- 👛 **کیف پول**: مدیریت کلیدهای عمومی/خصوصی
- 🌐 **شبکه P2P**: پایه‌های شبکه توزیع شده
- 🖥️ **رابط کاربری**: CLI تعاملی فارسی
- 🔒 **امنیت**: رمزنگاری SHA-256 و secp256k1

## 🛠️ نصب و راه‌اندازی

### پیش‌نیازها

```bash
# نصب Rust
```



## 🔧 اجزای فنی

### بلاک (Block)
- **Index**: شماره بلاک در زنجیره
- **Timestamp**: زمان ایجاد بلاک
- **Transactions**: لیست تراکنش‌ها
- **Previous Hash**: هش بلاک قبلی
- **Hash**: هش فعلی بلاک
- **Nonce**: عدد استفاده شده در Proof of Work
- **Merkle Root**: ریشه درخت مرکل تراکنش‌ها

### تراکنش (Transaction)
- **ID**: شناسه یکتا
- **From/To Address**: آدرس فرستنده و گیرنده
- **Amount**: مقدار انتقال
- **Timestamp**: زمان ایجاد
- **Signature**: امضای دیجیتال

### کیف پول (Wallet)
- **Private Key**: کلید خصوصی (32 بایت)
- **Public Key**: کلید عمومی
- **Address**: آدرس کیف پول (مشتق از کلید عمومی)

## 🔐 امنیت

- **هش‌سازی**: SHA-256 برای تمام محاسبات هش
- **امضای دیجیتال**: secp256k1 (همان Bitcoin)
- **Proof of Work**: الگوریتم اجماع برای جلوگیری از تقلب
- **اعتبارسنجی**: بررسی کامل زنجیره بلاک
